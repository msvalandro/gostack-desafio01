# Desafio 01: Conceitos do NodeJS

Crie uma aplicação para armazenar projetos e suas tarefas do zero utilizando [Express](https://expressjs.com/).

### Running

```sh
#install dependencies
yarn

#start application at http://localhost:3333
yarn dev
```

### Environment

- Node.js v12.14.1
- PostgreSQL 12.1
